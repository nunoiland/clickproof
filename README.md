# ClickProof - 링크 검문소

> 클릭하기 전에 확인하세요. URL 안전성을 실시간 분석하여 피싱, 스미싱, 악성 사이트로부터 보호합니다.

---

## 이런 분들에게 필요합니다

- 카카오톡, 문자로 받은 링크가 안전한지 확인하고 싶은 분
- 피싱 사이트에 개인정보를 입력할까 걱정되는 분
- 가족, 직장 동료에게 안전한 인터넷 사용 습관을 알려주고 싶은 분

---

## 주요 기능

### Chrome 확장 프로그램
- **13개 항목 URL 안전도 분석** — 주소창 옆 뱃지로 안전/주의/위험을 한눈에 확인
- **위험 사이트 자동 차단** — 위험 URL 접속 전 경고 페이지 표시
- **한국형 피싱 감지** — 네이버, 카카오, 은행 등 한국 주요 서비스 사칭 탐지
- **스미싱 키워드 자동 감지** — 택배 사칭, 정부 사칭, 결제 사칭 등 패턴 인식
- **금융기관 도메인 인증** — 공식 금융 사이트 여부 즉시 확인
- **검색 결과 안전 뱃지** — Google, Naver 검색 결과에 안전도 표시
- **이메일 링크 보호** — Gmail, Naver 메일 내 위험 링크 하이라이트
- **인앱 브라우저 감지** — 카카오톡, 네이버 등 앱 내 브라우저 경고
- **단축 URL 경고** — bit.ly 등 단축 URL 클릭 시 확인 요청

### 보안과 프라이버시
- **100% 로컬 분석** — 모든 URL 분석이 사용자 기기에서 수행됩니다
- **외부 서버 전송 없음** — 방문 기록, 개인정보 등 어떠한 데이터도 외부로 전송하지 않습니다
- **최소 권한** — 필요한 최소한의 브라우저 권한만 사용합니다

## 분석 항목

| 항목 | 설명 |
|------|------|
| HTTPS 검증 | 암호화 연결 여부 확인 |
| 타이포스쿼팅 | 공식 도메인과 유사한 사칭 도메인 감지 |
| 단축 URL | 실제 목적지를 숨기는 단축 URL 감지 |
| 도메인 길이 | 비정상적으로 긴 도메인 감지 |
| 서브도메인 깊이 | 과도한 서브도메인 사용 감지 |
| 특수문자 | URL 위장용 특수문자 사용 감지 |
| IP 직접 접속 | 도메인 없이 IP로 접속하는 경우 감지 |
| 의심 경로 | /login, /verify 등 피싱에 자주 사용되는 경로 감지 |
| 무료 호스팅 | 무료 호스팅 서비스 사용 여부 확인 |
| 호모그래프 공격 | 키릴/그리스 문자를 이용한 도메인 위장 감지 |
| Punycode | 국제화 도메인 인코딩 위장 감지 |
| TLD 신뢰도 | 의심스러운 최상위 도메인 확인 |
| 한국 신뢰 도메인 | 한국 주요 서비스 및 금융기관 공식 도메인 인증 |
