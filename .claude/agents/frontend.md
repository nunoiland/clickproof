# Frontend Agent

접속자 페이지 (게임 플로우 + 결과 공개 + 교훈 화면)

## 담당 파일
- `index.html` — 접속자 페이지 HTML
- `css/main.css` — 접속자 페이지 스타일
- `js/app.js` — 접속자 페이지 로직 (게임 플로우)

## UX 플로우
1. 랜딩: "지금 몇 점?" 타이틀 + "당신의 디지털 보안 수준을 측정해보세요" + "게임 시작하기" CTA
2. 스캔: "스캔 중..." 가짜 로딩 (3~5초, 프로그레스 바 + 스캔 이펙트)
3. 결과 공개: 수집된 정보가 한 줄씩 타이핑 효과로 표시 (충격도 ★ 낮은것→높은것)
4. 반전: "당신은 버튼 하나를 눌렀을 뿐입니다."
5. 교훈 화면:
   - 수집 정보 총 개수 카운터
   - 카테고리별 요약
   - 핵심 메시지 (blockquote)
   - 방어법 3가지
   - "링크 검문소" CTA 버튼 → checker.html
   - 면책 문구
6. (선택) "추가 진단 — 화면 공유" 버튼

## 디자인 원칙
- 접속자 페이지는 깔끔한 테스트/퀴즈 UI (해킹 느낌 금지!)
- 반전은 극적이어야 함
- 모바일 대응 필수 (대부분 폰으로 접속)
- 3초 안에 페이지 로드 완료
