ClickProof — 링크 검문소 개인정보처리방침

시행일: 2026년 3월 25일

핵심 요약: ClickProof는 사용자의 브라우징 데이터를 수집, 저장, 전송하지 않습니다. 모든 URL 분석은 사용자의 기기에서 로컬로 수행됩니다.

1. 수집하는 개인정보

ClickProof는 어떠한 개인정보도 수집하지 않습니다.

  • 방문한 웹사이트 URL을 외부 서버로 전송하지 않습니다.
  • 브라우징 기록을 저장하지 않습니다.
  • 사용자를 식별할 수 있는 정보를 수집하지 않습니다.
  • 쿠키, 추적 코드, 분석 도구를 사용하지 않습니다.

2. URL 분석 방식

ClickProof의 URL 안전도 분석은 다음과 같이 수행됩니다:

  • 모든 분석은 로컬에서 수행: URL 구조 분석, 타이포스쿼팅 감지, 도메인 신뢰도 검사 등 모든 검사가 사용자의 브라우저 내에서 실행됩니다.
  • 외부 서버 통신 없음: 분석 과정에서 어떠한 외부 서버와도 통신하지 않습니다.
  • 내장 데이터베이스: 한국 주요 도메인 목록, 금융기관 화이트리스트 등은 확장 프로그램 내에 내장되어 있습니다.

3. 로컬 저장소 사용

ClickProof는 Chrome의 로컬 저장소(chrome.storage.local)에 다음 설정만 저장합니다:

  • 자동 보호 기능 ON/OFF 설정
  • 사용자가 허용한 도메인 목록 (경고 표시 안 함으로 선택한 사이트)

이 데이터는 사용자의 기기에만 저장되며 외부로 전송되지 않습니다.

4. 권한 사용 목적

  • activeTab: 현재 탭의 URL을 읽어 안전도를 분석합니다.
  • webNavigation: 페이지 이동 시 URL을 분석하여 위험 사이트를 사전에 차단합니다.
  • storage: 사용자 설정(자동 보호 ON/OFF, 허용 목록)을 로컬에 저장합니다.

5. 제3자 제공

ClickProof는 어떠한 데이터도 제3자에게 제공, 판매, 공유하지 않습니다.

6. 아동 개인정보

ClickProof는 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
